Get started with JKCSS
JKCSS CSS - Framework is underdeveloping. This Framework only contains CDN (Content Delivery Network or Content Distribution Network) and Downloaded CSS will be in future versions.
  1. Create a file called index.html copy and paste the following content to that file.
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>JKCSS DEMO</title>
</head>
<body>

<!-- You can customize following content -->
<h1>Hello World, I am JKCSS..!</h1>

</body>
</html>
  1. Now include JKCSS styles using CDN. Now you have to Update index.html as following
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>JKCSS DEMO</title>
<link rel="stylesheet" href=""> </head>
<body>

<!-- You can customize following content -->
<h1>Hello World, I am JKCSS..!</h1>

</body>
</html>
